Chương trình Restful API Service Question & Answer được viết bằng :
- Python3
- Django Rest Framework

Thành phần cấu trúc của chương trình :
- API Question cho phép người dùng tạo câu hỏi (status True là đóng, False là mở)
- API Answer cho phép người dùng xam câu hỏi và tạo câu trả lời
- Người dùng cần đăng ký tài khoản và đăng nhập để đăng câu hỏi và tạo câu trả lời
- Nhập số pk sau hai đường dẫn api/questions/[pk] và api/answers/[pk] để RUD (xem ,sửa, xóa)
- API Question và API Answer có upvote, downvote và tag
- API Register cho phép người dùng đăng ký tài khoản
- API Login cho phép người dùng đăng nhập tài khoản (permission dựa trên token)
- API Logout cho phép người dùng đăng xuất (gửi post để logout)
- API Tags cho phép người dùng thêm tag

Tất cả các đường dẫn của chương trình :
api/questions/
api/answers/
api/tags/
api/token/
api/token/refresh/
rest-auth/register/
rest-auth/login/
rest-auth/logout/
vote/
user/
